# Changelog

## [Unreleased]

## [0.1.1] - 2026-04-08

### Added
- 画面下部にステータスバーを追加し、作業状態と `ffprobe` の利用可否を常時表示するようにした
- `設定` メニューに `会社/作品情報をリセット` を追加し、保存済みの会社名、作品名、候補一覧、前回の書き出し先、差出人フッターをローカル設定から削除できるようにした

### Changed
- 読み込み準備中、動画検索中、解析中、フレーム数精査中の進捗表示をステータスバーへ集約した

### Fixed
- Windows でドラッグアンドドロップ時に `ffprobe.exe` 起動に伴うコマンドプロンプトが大量に表示される問題を修正した
